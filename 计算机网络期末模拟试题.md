# 参考答案

### 一、名词解释（要点）

**1. 网络体系结构**：计算机网络各分层模型的层次及其对应全部协议的集合。典型模型包括TCP/IP四层模型、OSI七层模型，每层配套专属协议。

**2. 网络协议**：为实现计算机网络间数据交换而人为制定的规则、约定与标准，由语法、语义、同步三大要素组成。

**3. 网桥与交换机**：网桥工作在数据链路层，依据目的MAC地址查询交换表，对帧执行过滤或转发。交换机是多端口网桥，原理一致。

**4. 路由器**：工作在网络层的核心互联设备，功能为路由选择与分组存储转发，实现异构网络互联。

**5. RIP协议**：路由信息协议，内部网关协议，基于距离向量算法的动态路由选择协议。

**6. OSPF协议**：开放式最短路径优先协议，内部网关协议，基于链路状态算法的动态路由选择协议。

**7. VPN**：以公用互联网为传输载体，通过隧道加密技术将分散私网互联形成的逻辑专用网络。

**8. NAT**：内网主机访问互联网时，自动将私有IP转换为公网全球IP的技术，分传统NAT和NAPT两种。

**9. TPDU**：运输层对等实体间传输数据的基本单元，TCP的TPDU为TCP报文段，UDP的TPDU为UDP报文。

**10. URL**：互联网中标识和访问网络资源的标准格式，是定位网络资源的通用方法。

---

### 二、简答题（要点）

**1. 网络协议定义及三要素：**
- 定义：为实现计算机网络数据交换制定的规则、约定与标准。
- 语法：数据/控制信息的结构、编码格式。
- 语义：报文包含的控制信息、数据含义及接收方响应动作。
- 同步：通信双方时序、速率、差错处理等匹配规则。

**2. 五层协议体系结构（自下而上）：**

| 层次 | 传输单位 | 典型设备 |
|------|----------|----------|
| 物理层 | 比特 | 中继器、集线器Hub |
| 数据链路层 | 帧 | 网桥、交换机 |
| 网络层 | 分组（包） | 路由器 |
| 运输层 | 报文 | 网关 |
| 应用层 | 报文 | 网关 |

**3. 物理层接口四大特性：**
- 机械特性：连接器外形、尺寸、引脚排布、固定装置。
- 电气特性：接口电压、电流大小、电平标准。
- 功能特性：每个引脚对应的信号功能。
- 过程特性：设备通信时接口信号交互的时序流程。

**4. TCP/IP核心思想：**
- 核心目标：实现异构网络互联。
- 屏蔽底层各类物理网络硬件与协议差异，将全网抽象为统一虚拟网络。
- 依靠整套TCP/IP协议簇完成跨网络数据传输。

**5. 网络基础：**
- 两大组成：通信子网（负责数据传输通信）、资源子网（负责数据处理与用户资源）。
- OSI七层（自下而上）：物理层→数据链路层→网络层→运输层→会话层→表示层→应用层。
- 三种交换技术：电路交换（电话网）、报文交换、分组交换（现代网络主流）。

**6. IP vs UDP可靠性差异：**
- 均为无连接、尽最大努力交付的不可靠协议。
- IP：仅对IP首部做校验和检测，数据部分不校验。
- UDP：构造12字节伪首部，对整个UDP报文（首部+数据）完整校验。

### 三、计算题（答案）

**1. CRC循环冗余校验**
- (1) G(x) = x⁴ + x + 1 → 二进制除数：**10011**
- (2) 被除数 = 原始数据 1101011011 后补4个0 → **11010110110000**

模2除法过程：
```
  11010110110000
⊕ 10011
  —————————————
   010011
  ⊕ 10011
    ———————
    000001011
       ...
```
- 余数 = **1110**，即CRC冗余码 = **1110**
- (3) 发送序列 = 原始数据 + CRC冗余码 = **1101011011 1110**

**2. CIDR地址块计算：**
- (1) 子网掩码：/20 → **255.255.240.0**
- (2) 最小地址：**192.168.64.0**，最大地址：**192.168.79.255**
- (3) IP总数 = 2¹² = **4096** 个
- (4) 等效C类网数 = 4096 ÷ 256 = **16** 个

**3. C类子网划分：**
- (1) 每子网≥30台，2⁵−2=30，主机位 n=5；子网位 = 8−5 = **3位**
- (2) 子网掩码：**255.255.255.224**（/27）
